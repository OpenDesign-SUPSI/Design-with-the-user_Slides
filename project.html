How we will work together #01

  1. Two groups of 4 students: A and B
  2. Form the groups
  3. Send one e-mail per group to info@openp2pdesign.org with, for each member of the group:
    1. Name - Surname
    2. GitHub username (not the password!)
    3. e-mail address

How we will work together #03

  1. Choose a project:
    1. It can be from another course or already done
    2. ... but you need to be sure about your rights on an existing project
    3. The importance is on testing the process of working together in a distributed way, not on the complexity of the project
    4. Test/design hardware, software, design, ...
  2. Simple/small enough for fabbing it on the last day

How we will work together #04

  1. Use the group repository for files
  2. Use the group repository issues for discussion
  3. Add the licenses (for hardware, software, design, ...) as files + README.md
